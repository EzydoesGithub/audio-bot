	``https://www.youtube.com/watch?v=r1AtmY-RMyQ``

	``sudo apt install ffmpeg``

- Node JS & NPM

	Node JS & NPM Windows Installation:
	``https://www.youtube.com/watch?v=AuCuHvgOeBY``
	
	Node JS & NPM Ubuntu Installation:
	``sudo apt update``
	``sudo apt install nodejs npm``
